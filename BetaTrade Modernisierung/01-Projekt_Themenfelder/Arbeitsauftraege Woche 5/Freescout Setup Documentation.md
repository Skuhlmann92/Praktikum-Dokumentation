# Freescout Setup Documentation

## 1.0 Freescout Erkunden

Logge dich als Admin in Freescout ein und schau dich erstmal ein wenig um, um dich mit der Anwedung vertraut zu machen. Es sind bereits ein paar User Accounts und Mailboxen vorhanden.

### 1.1 Fehlerhafte Konfiguration anpassen
Schaue dir die Connection Settings der vorhandenen Mailboxen in Freescout an, dort scheint ein Konfigurationsproblem vorzuliegen, weshalb keine Mails versendet oder erhalten werden können. Connection Test funktioniert nur, wenn die Mailboxen auf Mailcow und in Freescout existieren und kommunizieren können!

### 1.2 User Accounts erstellen
Damit sich jeder Betatrade-Mitarbeiter auch bei Freescout anmelden kann benötigen diese eigene User Accounts. Erstelle noch 2-3 weitere User Accounts

Role: User
Email: <Nutzername>@ticket.betatrade.beta
Passwort: BetaTrade@TQ3b! (Haken entfernen bei Send an invite Email)
Füge dabei jeden User zur richtigen Mailbox hinzu.

### 1.3 Mailboxen erstellen
Erstelle für die neu erstellten User Accounts Mailboxen, orientiere dich dafür an den bereits vorhandenen Mailboxen. Teste jeweils die Verbindung.
Zugangsdaten bei Sending und Fetching Emails sind jeweils die E-mail + Standardpasswort
Logge dich nach erstellen aller Accounts bei 1-2 Usern ein und versende eine Testmail an support@betatrade.beta
